# AI News Email Agent

An AI-powered automation workflow that fetches news from external APIs, summarizes articles using OpenAI, and sends daily email digests automatically.

## Features

- Fetches latest AI news via REST APIs
- Generates concise summaries using OpenAI API
- Automates workflows using n8n
- Sends daily email reports via SMTP
- Error handling and retry logic

## Tech Stack

- Python
- OpenAI API
- n8n
- REST APIs
- SMTP
- JSON

## Workflow

1. Fetch AI news from APIs
2. Process and clean news data
3. Generate summaries using OpenAI
4. Create email digest
5. Send automated email
